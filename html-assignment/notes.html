What Have I Learned

A basic idea of some tags i learned during the course

Paragraphs
The HTML tag for this is a "p" inside of "<" & ">"
This creates a basic paragraph and is usually placed inside the head or body
Headers
The HTML tag for this is "h" followed by a number value between 1-6 (h1-h6)
and is between a "<" and ">", to close the tag add a "/" before the "h"
The number value represents the size of the letters.
Emphasis or Italic's
The HTML to add this effect to a peice of text is "em" between
a "<", ">". To close the effect add a "/" before the "em". Done
correctly and the Effect should look like this!
Highlights
The HTML tag for this is "mark" between a "<", ">". To close the
effect add a "/" before the "mark". The resulting effect should look
like this
Organized Lists
The HTML tag for this is "ol" between a "<", ">". To close the
effect add a "/" before the "ol". To add content to the list
use the same format as with the opening and closing tag using but with
"li" instead of "ol"
